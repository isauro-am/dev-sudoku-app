# sudoku

Simple sudoku game made with flutter.
( No ads, no in-app purchases, no tracking, no analytics, no data collection )

## Game Modes

- Easy
- Medium
- Hard

## Features

- game modes
  - Input Mode
  - Clue Mode
  - Note Mode
